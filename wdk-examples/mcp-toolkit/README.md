# MCP Toolkit Examples

| Example | Description |
|---------|-------------|
| [Basic Server](./basic-server.ts) | Create a WDK MCP server with wallet, pricing, indexer, swap, bridge, lending, and fiat tools |
| [LangChain Agent (TypeScript)](./langchain/typescript/agent.ts) | Connect a TypeScript LangChain agent to the WDK MCP server |
| [LangChain Agent (Python)](./langchain/python/agent.py) | Connect a Python LangChain agent to the WDK MCP server |
