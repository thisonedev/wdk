# EVM Examples

Examples for the [`@tetherto/wdk-wallet-evm`](https://github.com/tetherto/wdk-wallet-evm) package.

| Example | Description |
|---------|-------------|
| [Create Wallet](./create-wallet.ts) | Initialize a wallet manager and derive accounts from a seed phrase |
| [Manage Accounts](./manage-accounts.ts) | Work with multiple accounts and custom derivation paths |
| [Check Balances](./check-balances.ts) | Query native token and ERC-20 balances for owned accounts |
| [Read-Only Account](./read-only-account.ts) | Monitor balances for any address without a private key |
| [Send Transaction](./send-transaction.ts) | Estimate fees and send native token transactions (EIP-1559 + legacy) |
| [Token Transfer](./token-transfer.ts) | Transfer ERC-20 tokens and estimate transfer fees |
| [Sign & Verify Message](./sign-verify-message.ts) | Sign messages and verify signatures |
| [Fee Management](./fee-management.ts) | Retrieve current network fee rates |
| [Memory Management](./memory-management.ts) | Securely dispose wallets and clear private keys from memory |
