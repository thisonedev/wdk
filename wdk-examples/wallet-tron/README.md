# TRON Examples

Examples for the [`@tetherto/wdk-wallet-tron`](https://github.com/tetherto/wdk-wallet-tron) package.

| Example | Description |
|---------|-------------|
| [Create Wallet](./create-wallet.ts) | Initialize a wallet manager and derive TRON accounts from a seed phrase |
| [Manage Accounts](./manage-accounts.ts) | Work with multiple accounts and custom BIP-44 derivation paths |
| [Check Balances](./check-balances.ts) | Query native TRX and TRC20 token balances for owned accounts |
| [Read-Only Account](./read-only-account.ts) | Monitor balances for any TRON address without a private key |
| [Send Transaction](./send-transaction.ts) | Estimate fees and send native TRX transactions |
| [Token Transfer](./token-transfer.ts) | Transfer TRC20 tokens and estimate transfer fees |
| [Sign & Verify Message](./sign-verify-message.ts) | Sign messages and verify signatures |
| [Fee Management](./fee-management.ts) | Retrieve current network fee rates |
| [Memory Management](./memory-management.ts) | Securely dispose wallets and clear private keys from memory |
