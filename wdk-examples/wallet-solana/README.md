# Solana Examples

Examples for the [`@tetherto/wdk-wallet-solana`](https://github.com/tetherto/wdk-wallet-solana) package.

| Example | Description |
|---------|-------------|
| [Create Wallet](./create-wallet.ts) | Initialize a wallet manager and derive Solana accounts from a seed phrase |
| [Manage Accounts](./manage-accounts.ts) | Work with multiple accounts and custom SLIP-0010 derivation paths |
| [Check Balances](./check-balances.ts) | Query native SOL and SPL token balances for owned accounts |
| [Read-Only Account](./read-only-account.ts) | Monitor balances for any Solana address without a private key |
| [Send Transaction](./send-transaction.ts) | Estimate fees and send native SOL transactions |
| [Token Transfer](./token-transfer.ts) | Transfer SPL tokens and estimate transfer fees |
| [Sign & Verify Message](./sign-verify-message.ts) | Sign messages and verify signatures |
| [Fee Management](./fee-management.ts) | Retrieve current network fee rates |
| [Memory Management](./memory-management.ts) | Securely dispose wallets and clear private keys from memory |
