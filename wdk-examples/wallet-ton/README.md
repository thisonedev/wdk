# TON Examples

Examples for the [`@tetherto/wdk-wallet-ton`](https://github.com/tetherto/wdk-wallet-ton) package.

| Example | Description |
|---------|-------------|
| [Create Wallet](./create-wallet.ts) | Initialize a wallet manager and derive TON accounts from a seed phrase |
| [Manage Accounts](./manage-accounts.ts) | Work with multiple accounts and custom BIP-44 derivation paths |
| [Check Balances](./check-balances.ts) | Query native TON and Jetton balances for owned accounts |
| [Read-Only Account](./read-only-account.ts) | Monitor balances for any TON wallet without a private key |
| [Send Transaction](./send-transaction.ts) | Estimate fees and send native TON transactions |
| [Token Transfer](./token-transfer.ts) | Transfer Jettons and estimate transfer fees |
| [Sign & Verify Message](./sign-verify-message.ts) | Sign messages and verify signatures |
| [Fee Management](./fee-management.ts) | Retrieve current network fee rates |
| [Memory Management](./memory-management.ts) | Securely dispose wallets and clear private keys from memory |
