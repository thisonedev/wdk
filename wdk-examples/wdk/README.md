# WDK Core Examples

Examples for the [`@tetherto/wdk`](https://github.com/tetherto/wdk) package.

| Example | Description |
|---------|-------------|
| [Getting Started](./getting-started.ts) | Generate a seed phrase, validate it, and create a WDK instance |
| [Register Wallets](./register-wallets.ts) | Register Solana, TON, and TRON wallet managers in one WDK instance |
| [Manage Accounts](./manage-accounts.ts) | Retrieve accounts by index and path and inspect multi-chain balances |
| [Send Transactions](./send-transactions.ts) | Quote and optionally send native transactions across multiple chains |
| [Middleware](./middleware.ts) | Register middleware and inspect account access hooks |
| [Error Handling](./error-handling.ts) | Handle missing registrations and dispose selected wallets safely |
