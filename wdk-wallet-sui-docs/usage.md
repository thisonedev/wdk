# Usage

Guide to using the `@tetherto/wdk-wallet-sui` module.

The `@tetherto/wdk-wallet-sui` module provides wallet management for SUI blockchain.

### [Getting Started](./usage/getting-started.md)
Install the package and create your first Sui wallet.

### [Manage Accounts](./usage/manage-accounts.md)
Work with multiple Sui accounts and custom derivation paths.

### [Check Balances](./usage/check-balances.md)
Query native SUI and coin balances.

### [Send Transactions](./usage/send-transactions.md)
Send native SUI and estimate transaction fees.

### [Transfer Tokens](./usage/transfer-tokens.md)
Transfer Sui-based tokens and manage transaction blocks.

### [Sign & Verify Messages](./usage/sign-verify-messages.md)
Sign messages and verify signatures with Sui accounts.

### [Error Handling](./usage/error-handling.md)
Handle Sui-specific errors and manage gas budgets.
